# Quiz App

An interactive quiz application built with vanilla JavaScript for practicing question flows, answer handling, and score display.

## Features

- Presents quiz questions with interactive answer choices.
- Tracks user progress and final score.
- Runs entirely in the browser with no backend dependency.
- Useful as a beginner project for working with arrays, events, and UI state.

## Tech Stack

- HTML
- CSS
- JavaScript

## Getting Started

- Clone or download this repository.
- Open index.html in your browser, or run the folder with VS Code Live Server.


## Notes

- This project is designed for local frontend use only.
- Open index.html to start the quiz.
